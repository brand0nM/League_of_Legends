# Pyeague_of_Legends
## Overview
Retrieve and store data for a players recent games, champions played, and ranked info.

### Purpose
Rest API's allow us to query data from Riot about Pyeague of Pegend based on region or summoner name. We can further create directories organize the data in JSON files.
